Dyson Sphere Program Dependency Graph

Details

This page renders a dependency graph for the complete set of resources and buildings in the game Dyson Sphere Program.

This was build using viz.js which is a Graphviz Javascript port using Emscripten. The panning and zooming is supported by svg-pan-zoom.

Usage Guide

Pan: hold left-click and drag
Zoom: scroll or double-click
Filter: remove nodes using the "Filter" regex input below

Filters

Nodes matching "Filter" are removed. Lines NOT matching "Select" are also removed.

If a node gets past all filters it will make adjacent nodes visible too.

A number of #tags can also be used including #raw, #rare, #junk, #logistics, #building, #t1 - #t7, #bt1 - #bt4
